owo)/

Requirement:
```
Pillow
```



If you never install Pillow before, please use `pip` or `easy_install` to install it first
```
$ pip install Pillow

or

$ easy_install Pillow
```

Then, just put convert.py under the folder you want to execute.

Now, you can execute it!

```
$ python convert.py
```
